# Vue Canvas
VueJS + PaperJS to create a simple, powerful canvas

<img src="static/final.png" alt="final image" width="1000px">

## Set Up
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
